## 如何使用

### 查看检查点详情

点击检查点 ID 查看完整的 JSON 文件，包含：
- Git 提交信息
- 代码统计
- 已完成任务
- 待办任务
- 环境配置
- 已知问题和解决方案

### 恢复到某个检查点

```bash
# 后端恢复
cd zsxq-api
git checkout <backend-commit>

# 前端恢复
cd zsxq-web
git checkout <frontend-commit>
```

### 创建新检查点

使用 Claude Code 指令：
```
/progress-save '检查点描述'
```
